คอมพิวเตอร์และอุปกรณ์ต่อพ่วง

1. ประเภทของคอมพิวเตอร์

คอมพิวเตอร์สามารถแบ่งประเภทได้หลากหลายวิธี ขึ้นอยู่กับเกณฑ์ที่ใช้ในการจำแนก เช่น ลักษณะการใช้งาน หรือความสามารถในการประมวลผล

การแบ่งตามลักษณะการใช้งาน

เป็นการจำแนกคอมพิวเตอร์ตามวัตถุประสงค์หลักที่ถูกออกแบบมาเพื่อใช้งาน

1. คอมพิวเตอร์ที่ใช้ในงานทั่วไป (General Purpose Computer)

เป็นคอมพิวเตอร์ที่สามารถทำงานได้หลากหลายรูปแบบ โดยผู้ใช้สามารถติดตั้งหรือเปลี่ยนโปรแกรม (Software) ได้อย่างอิสระ ทำให้เครื่องเดียวสามารถทำหน้าที่ได้หลายอย่างตามความต้องการ

คุณสมบัติหลัก:

  • ความยืดหยุ่นสูง: สามารถทำงานได้หลากหลายด้วยการเปลี่ยนซอฟต์แวร์
  • ใช้งานง่าย: มีระบบปฏิบัติการที่คุ้นเคย เช่น Windows, macOS หรือ Android
  • ฮาร์ดแวร์มาตรฐาน: ใช้ส่วนประกอบที่หาซื้อได้ทั่วไป

ตัวอย่าง:

  • คอมพิวเตอร์ตั้งโต๊ะ (Desktop PC)
  • คอมพิวเตอร์พกพา (Laptop)
  • สมาร์ตโฟน (Smartphone) และแท็บเล็ต (Tablet)

2. คอมพิวเตอร์ที่ใช้งานเฉพาะด้าน (Special Purpose Computer)

เป็นคอมพิวเตอร์ที่ถูกสร้างขึ้นเพื่อทำงานอย่างใดอย่างหนึ่งเท่านั้น โดยโปรแกรมหรือชุดคำสั่งจะถูกฝัง (Embedded) อยู่ในตัวเครื่องตั้งแต่ขั้นตอนการผลิต ทำให้ไม่สามารถเปลี่ยนแปลงได้ จึงมีชื่อเรียกว่า “สมองกลฝังตัว” (Embedded System)

คุณสมบัติหลัก:

  • ประสิทธิภาพสูงต่องานเฉพาะ: ทำงานที่ได้รับมอบหมายได้อย่างรวดเร็วและแม่นยำ
  • ขนาดเล็กและประหยัดพลังงาน: มักถูกติดตั้งในอุปกรณ์ต่างๆ
  • ความเสถียรสูง: มีความทนทานและทำงานซ้ำๆ ได้โดยไม่ผิดพลาด

ตัวอย่าง:

  • เครื่องใช้ไฟฟ้าในบ้าน เช่น เครื่องซักผ้าอัตโนมัติ
  • เครื่องจักรในโรงงานอุตสาหกรรม เช่น เครื่อง CNC และระบบ PLC
  • ระบบควบคุมในยานยนต์ เช่น ระบบเบรก ABS

การแบ่งตามความสามารถในการประมวลผล

เป็นการจำแนกคอมพิวเตอร์ตามความเร็วในการประมวลผลและขนาดของเครื่องจากสูงสุดไปต่ำสุด

1. ซูเปอร์คอมพิวเตอร์ (Supercomputer)

เป็นคอมพิวเตอร์ที่มีประสิทธิภาพสูงสุดในโลก ใช้ในการคำนวณที่ซับซ้อนและใช้ข้อมูลจำนวนมหาศาล เช่น การจำลองสภาพอากาศ และการวิจัยทางวิทยาศาสตร์

2. เมนเฟรมคอมพิวเตอร์ (Mainframe Computer)

เป็นคอมพิวเตอร์ขนาดใหญ่ที่มีประสิทธิภาพรองลงมา ใช้เป็นศูนย์กลางในการประมวลผลข้อมูลในองค์กรขนาดใหญ่ และสามารถรองรับผู้ใช้งานจำนวนมากพร้อมกันได้

3. มินิคอมพิวเตอร์ (Minicomputer)

เป็นคอมพิวเตอร์ขนาดกลาง มีประสิทธิภาพสูงกว่าไมโครคอมพิวเตอร์ มักใช้เป็นเซิร์ฟเวอร์ในองค์กรขนาดกลาง และในงานวิศวกรรมเฉพาะทาง

4. ไมโครคอมพิวเตอร์ (Microcomputer)

คือคอมพิวเตอร์ที่เราใช้ในชีวิตประจำวัน มีขนาดเล็กที่สุดและราคาถูกที่สุด ใช้หน่วยประมวลผลที่เรียกว่าไมโครโพรเซสเซอร์

ตัวอย่าง:

  • คอมพิวเตอร์ตั้งโต๊ะ (Desktop PC)
  • คอมพิวเตอร์พกพา (Laptop)
  • สมาร์ตโฟน (Smartphone)

2. หลักการทำงานของคอมพิวเตอร์

การใช้ semantic Tag เช่น header, nav, section, article, footer ทำให้โค้ด HTML มีโครงสร้างที่ชัดเจนและอ่านง่ายขึ้น ซึ่งเป็นประโยชน์ต่อ SEO และการเข้าถึง

3. ปัจจัยในการทำงานของคอมพิวเตอร์

CSS3 มีคุณสมบัติใหม่ๆ เช่น การจัดการตัวอักษร การสร้างเงา และการเปลี่ยนแปลงที่ช่วยให้การออกแบบเว็บมีความสวยงามและตอบสนองได้ดีขึ้น

4. อุปกรณ์ต่อพ่วง

JavaScript เป็นภาษาที่ใช้ในการพัฒนาเว็บที่ช่วยให้สามารถสร้างฟังก์ชันการทำงานที่ซับซ้อนได้ เช่น การตรวจสอบข้อมูลแบบเรียลไทม์ การสร้างแอนิเมชัน และการจัดการ DOM
