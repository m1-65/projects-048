การประกอบเครื่องคอมพิวเตอร์

1. การเตรียมเครื่องมือและอุปกรณ์

ก. เครื่องมือที่จำเป็น

  • ไขควงแบบหัวแฉก (Phillips Screwdriver)
  • สายรัดข้อมือกันไฟฟ้าสถิต (Anti-static Wrist Strap)
  • แหนบหรือคีมปากแหลมขนาดเล็ก
  • ผ้าสะอาดหรือแปรงเล็ก
  • ไฟฉายหรือโคมไฟเล็ก

ข. อุปกรณ์คอมพิวเตอร์ที่ใช้ในการประกอบ

  • เมนบอร์ด (Mainboard / Motherboard)
  • ซีพียู (CPU - Central Processing Unit)
  • พัดลมและฮีตซิงก์ (CPU Fan / Heatsink)
  • แรม (RAM - Memory Module)
  • ฮาร์ดดิสก์ / SSD (Hard Disk Drive / Solid State Drive)
  • การ์ดจอ (Graphic Card)
  • พาวเวอร์ซัพพลาย (Power Supply Unit - PSU)
  • เคส (Case / Chassis)
  • สาย SATA / สายไฟต่าง ๆ
  • จอภาพ คีย์บอร์ด เมาส์ สำหรับทดสอบ

ค. สภาพแวดล้อมที่เหมาะสม

  • พื้นที่สะอาด ไม่มีฝุ่น
  • โต๊ะเรียบและมั่นคง
  • ไม่ควรประกอบบนพื้นพรม
  • มีแสงสว่างเพียงพอ

2. การติดตั้งชีพียูและชุดระบายความร้อน

📌 ขั้นตอนการติดตั้ง CPU

  • ตรวจสอบซ็อกเก็ตของเมนบอร์ดว่ารองรับกับซีพียู
  • เปิดตัวล็อกซ็อกเก็ต (ง้างก้านล็อกขึ้น)
  • จับซีพียูอย่างระมัดระวัง โดยจับเฉพาะขอบ
  • วางซีพียูโดยให้มุมลูกศรตรงกันกับซ็อกเก็ต
  • ล็อกก้านกลับลงให้แน่น

❄️ ขั้นตอนการติดตั้งชุดระบายความร้อน

  • ทาซิลิโคนบาง ๆ ตรงกลางซีพียู (ถ้าจำเป็น)
  • วางชุดระบายความร้อนลงอย่างตรงตำแหน่ง
  • กดหรือล็อกชุดระบายความร้อนให้แน่น
  • เสียบสายพัดลมเข้ากับช่อง CPU_FAN
  • ตรวจสอบความแน่นก่อนประกอบส่วนอื่นต่อ

3. การติดตั้งแผงหน่วยความจำ

📌 ความรู้เบื้องต้น

  • RAM คือหน่วยความจำหลักของเครื่องคอมพิวเตอร์
  • เลือกชนิดแรมให้ตรงกับเมนบอร์ด (DDR3 / DDR4 / DDR5)
  • การติดตั้งที่ถูกต้องช่วยให้ระบบทำงานแบบ Dual Channel ได้

🛠️ ขั้นตอนการติดตั้ง RAM

  • ตรวจสอบชนิดของ RAM ที่เมนบอร์ดรองรับ
  • ค้นหาช่องเสียบแรม (DIMM Slot)
  • เปิดก้านล็อกของช่องเสียบแรมทั้งสองด้าน
  • จับแผง RAM โดยไม่สัมผัสแผงวงจร
  • จัดตำแหน่งร่องให้ตรง แล้วกดลงไปจนแน่น
  • ตรวจสอบให้ก้านล็อกดีดกลับเข้าที่

4. การติดตั้งแผงวงจรหลัก

📌 ความรู้เบื้องต้น

  • Mainboard เป็นอุปกรณ์หลักที่ใช้เชื่อมต่อส่วนต่าง ๆ ของคอมพิวเตอร์
  • ต้องติดตั้งให้แน่นหนา ปลอดภัย และไม่สัมผัสกับโลหะโดยตรง

🛠️ ขั้นตอนการติดตั้ง Mainboard

  • ตรวจสอบขนาดเมนบอร์ดให้พอดีกับเคส
  • ใส่แผ่นฝาหลัง (I/O Shield) ที่ช่องด้านหลังเคส
  • ติดตั้งสกรูรอง (Standoff) ให้ตรงกับรูของเมนบอร์ด
  • วางเมนบอร์ดลงอย่างระมัดระวังบนสกรูรอง
  • ขันน็อตยึดเมนบอร์ดเข้ากับเคสให้แน่นพอดี
  • เชื่อมต่อสายต่าง ๆ จากเคสเข้ากับเมนบอร์ด (Power SW, Reset SW, LED, USB, Audio)

5. การต่อสายไฟเลี้ยงและเสียบสายสัญญาณ

📌 ความรู้เบื้องต้น

  • สายไฟเลี้ยงจ่ายพลังงานจากพาวเวอร์ซัพพลายไปยังเมนบอร์ดและอุปกรณ์
  • สายสัญญาณเชื่อมต่อข้อมูลระหว่างอุปกรณ์กับเมนบอร์ด
  • ต่อสายให้แน่นและถูกต้องช่วยให้ระบบทำงานได้อย่างเสถียร

🛠️ ขั้นตอนการต่อสายไฟและสายสัญญาณ

  • ต่อสายไฟหลัก 24 พิน (24-pin ATX) จาก PSU เข้ากับเมนบอร์ด
  • ต่อสายไฟ 4/8 พิน สำหรับจ่ายไฟซีพียู (CPU Power)
  • เสียบสาย SATA Power ไปยังฮาร์ดดิสก์หรือ SSD
  • เสียบสาย SATA Data จากฮาร์ดดิสก์/SSD ไปยังเมนบอร์ด
  • ต่อสายไฟพาวเวอร์ของการ์ดจอ (ถ้ามี) จาก PSU
  • ตรวจสอบและจัดสายไฟให้เรียบร้อย

6. การติดตั้งฮาร์ดดิสก์และออปติคัสไดรฟ์

📌 ความรู้เบื้องต้น

  • ฮาร์ดดิสก์และ SSD เป็นอุปกรณ์เก็บข้อมูลหลักของคอมพิวเตอร์
  • ออปติคัลไดรฟ์ใช้สำหรับอ่าน/เขียนแผ่นดิสก์ เช่น DVD หรือ Blu-ray
  • ติดตั้งไดรฟ์ให้มั่นคงและเชื่อมต่อสายสัญญาณอย่างถูกต้อง

🛠️ ขั้นตอนการติดตั้ง

  • เลือกช่องติดตั้งในเคสที่เหมาะสม (3.5 นิ้วสำหรับ HDD, 5.25 นิ้วสำหรับ Optical Drive)
  • วางไดรฟ์ลงในช่องและยึดด้วยสกรู
  • ต่อสาย SATA Data จากไดรฟ์ไปยังเมนบอร์ด
  • ต่อสาย SATA Power จาก PSU ไปยังไดรฟ์
  • ตรวจสอบความแน่นและความเรียบร้อยของสาย

7. การติดตั้งแผนการ์ด

📌 ความรู้เบื้องต้น

  • แผนการ์ดเพิ่มความสามารถให้คอมพิวเตอร์ เช่น การ์ดจอ การ์ดเสียง
  • ติดตั้งในสล็อต PCIe บนเมนบอร์ด
  • ติดตั้งอย่างระมัดระวังเพื่อป้องกันความเสียหาย

🛠️ ขั้นตอนการติดตั้ง

  • ปิดเครื่องและถอดปลั๊กไฟก่อนทุกครั้ง
  • เปิดฝาเคสและหาสล็อต PCIe ที่จะติดตั้งการ์ด
  • ถอดแผ่นปิดช่องสล็อตบนเคสออก
  • จับการ์ดที่ขอบ ไม่จับชิปหรือขาเชื่อมต่อ
  • ใส่การ์ดลงในสล็อตอย่างตรงและแน่น
  • ใช้น็อตยึดการ์ดกับเคส
  • ต่อสายไฟเสริมจาก PSU ถ้าการ์ดต้องการ
  • ปิดฝาเคสและเปิดเครื่องตรวจสอบการทำงาน

8. การตรวจสอบและจัดเก็บสาย

📌 ความรู้เบื้องต้น

  • จัดเก็บสายช่วยให้ภายในเคสดูเรียบร้อยและระบายความร้อนได้ดีขึ้น
  • ตรวจสอบการเชื่อมต่อสายไฟและสายสัญญาณให้ถูกต้อง

🛠️ ขั้นตอนการตรวจสอบและจัดเก็บสาย

  • ตรวจสอบการต่อสายไฟทุกจุดให้แน่นและถูกต้อง
  • ตรวจสอบสายสัญญาณ เช่น SATA, USB Front Panel, Audio Front Panel
  • ใช้สายรัด (Cable Ties) รัดสายไฟให้เป็นระเบียบ
  • จัดสายให้ไม่ขวางพัดลมหรือทางระบายความร้อน
  • เปิดเครื่องตรวจสอบการทำงานเบื้องต้นใน BIOS/UEFI
  • ปิดฝาเคสหลังจากตรวจสอบเรียบร้อย
