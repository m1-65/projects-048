อุปกรณ์ภายในเครื่องคอมพิวเตอร์

1. อุปกรณ์ภายในตัวเครื่อง

  • เมนบอร์ด (Mainboard)
  • ซีพียู (CPU - หน่วยประมวลผลกลาง)
  • แรม (RAM - หน่วยความจำชั่วคราว)
  • ฮาร์ดดิสก์ หรือ SSD (อุปกรณ์จัดเก็บข้อมูล)
  • การ์ดจอ (GPU - หน่วยประมวลผลกราฟิก)
  • พาวเวอร์ซัพพลาย (Power Supply)
  • ระบบระบายความร้อน / พัดลม
  • การ์ดเครือข่าย / การ์ดเสียง (ถ้ามี)

2. อุปกรณ์บนแผ่นเมนบอร์ด

  • ซ็อกเก็ตซีพียู (CPU Socket): ช่องสำหรับติดตั้งหน่วยประมวลผลกลาง (CPU)
  • ชิปเซ็ต (Chipset): ควบคุมการสื่อสารระหว่าง CPU กับอุปกรณ์อื่น ๆ
  • สล็อตแรม (RAM Slots): สำหรับติดตั้งหน่วยความจำหลัก (RAM)
  • ช่องเสียบการ์ดจอ (PCIe x16 Slot): สำหรับติดตั้งการ์ดจอ (GPU)
  • สล็อตขยายอื่น ๆ (PCIe x1, M.2): สำหรับอุปกรณ์เสริม เช่น SSD, การ์ดเสียง
  • พอร์ต SATA: ใช้เชื่อมต่อกับ HDD, SSD และไดรฟ์ DVD
  • ชิป BIOS / UEFI: เฟิร์มแวร์ที่เริ่มทำงานก่อนระบบปฏิบัติการ
  • พอร์ต I/O ด้านหลัง: เช่น USB, HDMI, LAN, แจ็คเสียง
  • พอร์ตภายใน (Internal Headers): สำหรับเชื่อมต่อกับปุ่ม Power, USB หน้าเคส ฯลฯ
  • ช่องเสียบพัดลม (Fan Headers): สำหรับพัดลม CPU และเคส
  • หัวต่อไฟ (Power Connectors): สำหรับจ่ายไฟให้กับเมนบอร์ดและ CPU
  • แบตเตอรี่ CMOS: เก็บค่าการตั้งค่า BIOS และเวลา
